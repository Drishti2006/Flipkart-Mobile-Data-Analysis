# 📊 Flipkart Mobile Data Analysis

## 📖 Overview

This project analyzes mobile phone data from Flipkart to extract meaningful insights about pricing, ratings, and brand performance.

## 🎯 Objectives

* Analyze price distribution
* Identify top brands
* Study ratings and customer preferences
* Understand discount trends

## 🛠️ Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## 📊 Key Insights

* Most phones fall in the mid-range category
* Discounts significantly impact pricing
* High price does not guarantee high ratings
* Some budget phones offer excellent value

## 📈 Visualizations

* Price Distribution
* Rating Distribution
* Price vs Rating
* Brand Analysis
* Correlation Heatmap

## 🚀 Conclusion

This project highlights how pricing and ratings influence customer decisions in the Flipkart mobile market.

## 🔗 Dataset

Kaggle Flipkart Mobile Dataset
